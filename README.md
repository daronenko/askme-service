# Ask Service

## Getting Started

### MacOS & Linux

1. Create and activate virtual environment:

```shell
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```shell
pip3 install -r requirements.txt
```

3. Run project:

```shell
./manage.py runserver
```

4. Open the `localhost:8000` path in the browser
